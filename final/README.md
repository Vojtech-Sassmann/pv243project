# Running the project

Run wildfly
`$JBOSS_HOME/bin/domain.sh`

Deploy application
`$JBOSS_HOME/bin/jboss-cli.sh --file=deploy.cli`

Then navigate to url 
`localhost:8080/TACOS-rest/`
